# Start Here

If you are an agent entering this toolbox with no prior context:

## Default Path

1. open `toolbox_manifest.json`
2. pick the active package that matches the task
3. open that package's `tool_manifest.json`
4. use that package's MCP or CLI contract

## Package Selection

- use `final-tools/` for general local workspace analysis, structured patching, and Tkinter audits
- use `_app-journal/` for project notes, decisions, logs, or a shared human/agent journal
- use `_manifold-mcp/` for evidence bags, reversible retrieval, and text-to-graph workflows
- use `_ollama-prompt-lab/` for local prompt evals and Ollama model comparison

## Avoid By Default

- `legacy/`
- `intake/`

These are not the preferred operational surface.
