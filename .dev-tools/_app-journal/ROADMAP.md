# Roadmap

## Implemented Now

- standalone SQLite store
- project bootstrap
- journal write and query tools
- export tool
- MCP server
- Tkinter manager UI

## Recommended Next

- saved filters and views in the UI
- full-text search with SQLite FTS
- backlinks between journal entries
- file attachments and screenshots
- timeline and decision views
- multi-project dashboard

## Non-Negotiable Rule

Any new feature must stay usable through both MCP and the Tkinter manager.
