# 📝 Task Manager

A simple, clean task management application built with Python and Tkinter.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete
- ✅ Delete tasks
- ✅ Persistent storage (tasks saved to JSON file)
- ✅ Task statistics and progress tracking
- ✅ Clean, modern interface
- ✅ Keyboard shortcuts support

## Installation

1. Ensure Python 3.x is installed on your system
2. Open a terminal/command prompt
3. Navigate to the `task_manager` directory
4. Run the application:

```bash
python todo_manager.py
```

## Usage

- **Add Task**: Type in the input box and click "Add"
- **Complete Task**: Select a task and click "Complete"
- **Delete Task**: Select a task and click "Delete"
- **Clear All**: Click "Clear All" button
- **View Stats**: Click "Show Stats" button

## File Structure
